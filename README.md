## Run Locally

Clone the project

```bash
  git clone https://github.com/rio-rifaldi/ableton-clone.git
```

Go to the project directory

```bash
  cd ableton-clone
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm run dev
```
